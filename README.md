# pill_dispenser_app
A mobile app for prescription recognition, schedule management and dispensing reminders using django and react native. The app utilizes OCR technology to extract text from prescription images, allowing users to manage their medication schedules effectively. It also includes features for tracking medication intake and sending reminders to ensure adherence to prescribed regimens. It is attached to a pill dispenser. The app communicates with the dispenser via celery.

# Stack
- Django
- React Native
- Celery
- Redis
